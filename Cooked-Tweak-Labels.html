labelled :: (MonadTweak m, LabelConstrs lbl) => lbl -> m a -> m a Source #

Apply a tweak to a given transaction if it has a specific label. Fails if it does not.This can be useful to apply a tweak to any transaction in a trace using somewhere.

someEndpoint = do
  ...
  validateTxSkel' txSkelTemplate
     { txSkelLabels =
        [ "InitialMinting"
        , "AuctionWorkflow"
        , label SomeLabelType]
     }

someTest = someEndpoint & eveywhere (labelled SomeLabelType someTweak)
anotherTest = someEndpoint & somewhere (labelled SomeLabelType someTweak)

labelled' :: (MonadTweak m, LabelConstrs lbl) => lbl -> m a -> m () Source #

Similar to labelled, but does not fail when the label is not present, thus making this tweak suitable to be used with everywhere
