Description

This module provides a convenient framework to look through UTxOs and search relevant ones based on predicates. For instance, it makes it very convenient to gather all UTxOs at a certain address.

The type of UTxO searches * --

type UtxoSearch m a = MaybeT (ListT m) (TxOutRef, a) Source #

If a UTxO is a TxOutRef with some additional information, this type captures a "stream" of potential UTxOs.

runUtxoSearch :: Monad m => UtxoSearch m a -> m [(TxOutRef, a)] Source #

Given a UTxO search, we can run it to obtain a list of actual UTxOs.
